===============================================
    GEO Optimizer - 快速启动指南
===============================================

⚠️ 重要：必须先启动 Docker Desktop！

✅ 启动步骤（3步）：

1. 启动 Docker Desktop
   - 在开始菜单搜索 "Docker Desktop"
   - 点击打开
   - 等待右下角鲸鱼图标停止动画（1-2分钟）

2. 验证 Docker 已启动
   - 打开命令提示符（CMD）
   - 运行：docker ps
   - 看到表格输出即可

3. 运行启动脚本
   - 双击 start.bat
   - 等待 30-60 秒
   - 浏览器会自动打开

===============================================

🐛 常见问题：

问题：Docker Desktop 未运行
解决：启动 Docker Desktop，等待图标静止

问题：端口被占用
解决：运行 docker-compose down 然后重新启动

问题：服务未就绪
解决：等待 30-60 秒，检查 docker-compose ps
